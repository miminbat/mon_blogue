La popularité de PHP diminue-t-elle ? Le langage de script a atteint sa position la plus basse dans l'index TIOBE,

Le langage de programmation PHP, autrefois dominant dans le domaine du développement Web, a atteint son plus bas niveau dans l'index de popularité TIOBE.

Paul Krill, IDG NS (adapté par Jean Elyan)
Published 12 mars 2024
Share:

PHP (Hypertext Preprocessor) est un langage de script côté serveur très répandu sur lequel est basé de nombreuses ressources sur Internet, depuis les blogues aux sites et applications Web les plus populaires du monde. La première version de PHP a été lancée il y a 30 ans. Nous en sommes aujourd'hui à la version 8, sortie en novembre 2020, mais la version 7 reste la plus utilisée. Il fonctionne sur le moteur Zend, qui est l'implémentation la plus populaire, mais il en existe d'autres, comme parrot, HPVM (Hip Hop Virtual Machine), et Hip Hop. Autrefois très populaire, l'index TIOBE suggère que PHP connaît un déclin.

Un probable support tiers pour Java 8

Le rapport présente les réponses de 440 utilisateurs Java dans le monde entier, clients ou pas de Perforce. Les répondants étaient répartis dans 72 pays, dont 27 % aux États-Unis. L’enquête a été réalisée entre novembre 2023 et janvier 2024. La plupart des personnes interrogées étaient des développeurs ou des architectes Java. Concernant l’utilisation du langage, 11 % ont déclaré avoir déjà mis à niveau vers Java 21, la version la plus récente de Java standard, tandis que 24 % ont déclaré utiliser la version 8 et 18 %, version 11. « Étant donné qu'Oracle a mis fin au support de niveau Premier de Java 8 en mars 2022, le pourcentage d’utilisateurs de Java 8 laisse supposer que les entreprises obtiennent le support de fournisseurs tiers comme Amazon, Azul et OpenLogic », a déclaré Perforce. Perforce Software pense que l'adoption de Java 21 augmentera à mesure qu'Oracle accélèrera la fréquence des versions de support à long terme LTS du JDK, de tous les trois ans à tous les deux ans.

D’autres enseignements

Amazon Web Services est le fournisseur de cloud le plus populaire (31 %), suivi de Microsoft Azure (18 %). La part des personnes interrogées qui ont déclaré n'utiliser aucun fournisseur de cloud est tombée à 13 %, contre 21 % l'année dernière. Sur la partie IDE, JetBrains IntelliJ IDEA se classe en tête avec 41 % des répondants qui l'utilisent. Eclipse a pris la deuxième place avec 23 %. Microsoft Visual Studio Code arrive en troisième position avec 19 %. Enfin, 36 % des sondées ont déclaré utiliser Apache Tomcat comme principal serveur d'applications, tandis que 15 % ont déclaré utiliser JBoss/Wildfly.
